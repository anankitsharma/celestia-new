## SEO + voice review

### Voice-guide compliance

- [x] No "best" / "#1" / "free" claims (Play Store rules)
- [x] No competitor names (Co-Star, Pattern, etc.)
- [x] No platform mentions (iOS, Apple) — Play Store rule
- [x] Voice register: Cup of Jo / NYT — verified by manual read
- [x] Loss-frame avoided in description (description = pre-conversion; loss-frame is for trial-end push only)
- [x] No psychic / tarot / fortune-teller terms
- [x] Unity language present: "for people who do their inner work", "for the questioners, not the believers"

### What's intentionally aggressive

- **"Not horoscopes. Not predictions. Not 'Capricorns are stubborn.'"** — opens with what we're NOT, audience-filters fast
- **"Astrology that doesn't insult your intelligence"** — direct positioning against generic horoscope apps
- **"For the questioners, not the believers"** — explicit unity / audience-naming
- **NASA JPL DE-441 callout** — explicit authority signal that competitors don't make

These are intentional choices to filter audience. They will turn off horoscope-app users (which is fine — they don't convert anyway).

### What's intentionally restrained

- No emoji in body text (only ✦ glyphs in features list)
- No "ASTROLOGY APP OF THE YEAR" or other claim
- No celebrity testimonials
- No ALL CAPS in body (only section headers)
- No exclamation marks
- No urgency / scarcity language

These are voice-guide rules. The description should read like a thoughtful product rather than an ad.

## Variants for A/B testing (post-launch)

Once you have ≥1,000 installs/month, A/B test description variants via Play Console's Listing Experiments feature. Suggested variants:

### Variant A (control) — the recommended copy above

### Variant B — methodology lead
```
Built using NASA JPL DE-441 ephemeris — the same astronomical standard
astronomers use. Then AI tuned to Western astrology traditions writes you
a real reading from your actual transits.

For people who want astrology that respects their intelligence.
```
*(Hypothesis: methodology-first leads convert better with skeptic-leaning users)*

### Variant C — audience-identity lead
```
This is not a horoscope app.

This is your real birth chart, your actual transits, your specific
questions — all written by AI tuned to the astrology you didn't grow
up with: Hellenistic foundations, modern psychological depth.

For the questioners, not the believers.
```
*(Hypothesis: identity-first leads convert better when paired with unity audience)*

### Variant D — feature-focused (control benchmark)
Strip the audience framing entirely; lead with bullet features. *(Hypothesis: this loses; tests whether positioning beats features)*

Run each variant for 2 weeks minimum at 25% traffic share. Winning variant becomes the new control.

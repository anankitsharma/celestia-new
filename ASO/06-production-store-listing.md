## FIELD-BY-FIELD REFERENCE

| Field | Limit | Indexed? | Conversion Impact |
|-------|-------|----------|-------------------|
| App Title | 30 chars | YES (highest weight) | Shows in search results |
| Short Description | 80 chars | YES (high weight) | Shows below title before install |
| Full Description | 4,000 chars | YES (medium weight) | Shows on listing page |
| What's New | 500 chars | Partially | Replaces short desc for existing users |
| Tags | 5 max | YES (discovery) | Feeds Browse/Explore recommendations |
| Category | 1 primary | YES | Determines category ranking |
| Developer Name | 50 chars | YES (low) | Trust signal |

### What's BANNED (will get rejected):
- Title/Developer name: "Free", "No Ads", "#1", "Best", "Top Rated", "Award"
- Title/Short desc: Emojis, emoticons, repeated special characters
- ALL CAPS in title (unless actual brand name)
- Performance/ranking claims anywhere
- Unattributed testimonials in description
- Keyword stuffing / word block lists

## 1. APP TITLE

```
Celestia: AI Astrology Guide
```

**Character count:** 28 / 30 (verified)

**Keywords captured:**
| Keyword | Search Volume | Competition | Why It's Here |
|---------|--------------|-------------|---------------|
| AI | Growing fast | LOW in astrology | Blue ocean — differentiator |
| Astrology | Very High | Very High | Category anchor — must be present |
| Guide | Medium | LOW | Warm intent ("guide me") vs cold ("tool") |

**Why this title wins for cold start:**
- "AI Astrology" is a compound keyword that matches growing search trends but has low competition among Western astrology apps
- "Guide" differentiates from the 50+ apps using "Horoscope" — fewer competitors, same user intent
- "Celestia" is memorable, searchable, no trademark conflicts
- Reads naturally: it IS an AI astrology guide

**What we sacrifice (and why it's OK):**
- "Horoscope" — highest raw volume, but INSANE competition. We capture this in short desc instead
- "Birth Chart" — captured in short desc and long desc
- "Daily" — captured in long desc
